# multi-agent-playground

## 1. Requirements
- python >= 3.10
## 2. Install
```Bash
pip install requirements.txt
```
## 3. Create Env File
```Bash
cp .env-example .env
```
- modify .env file
## 2. Run
- Open Visual Studio Code
- Open `Run and Debug` Menu
